Get Started with Workday

Welcome to the Workday Community. Below you’ll find all the resources needed for an introduction to the Workday service and release process.

Learn How Workday Delivers Features, including what's in a Service Update versus a Feature Release. You’ll also find out about Single Codeline development and the Workday Web Services Release Cycle.

With the introduction of each new release or feature, Workday publishes documentation to help walk you through deployment. Visit About Workday Documentation for an in-depth look the types of content accompanying an update, including publication schedules and best practices.

Whether you’re in the middle of your initial Workday deployment, or adding more features in Production, the Welcome to Workday and Deploy Workday resources help you prepare for a seamless deployment.

As you near the end of your initial Workday deployment, you’re probably wondering what your next steps are. Use the Move to Production resources to help get ready to go live and manage Workday.

Get the most out of your implementation with free training resources on key Workday functions, including:

  • Business Process
  • Configurable Security
  • Core Concepts
  • Report Writer
  • Workday Integration
  • Workday Web Services

Once you’re done with the resources above, learn how to Get Started on the Workday Community.
